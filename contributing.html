HowTo add a post to WASM.NEWS

This page is hosted on GitHub using the static site generator jekyll. Posts are written as markdown files (.md). If you want to post something on WASM.NEWS you should make a pull request with your post that could easily be merged.

Just head to the post folder in the repository, click on New file and enter a name matching the scheme: YYYY-MM-DD-Title-of-the-post.md

Then add the necessary informations to the file, following the instructions below and click on Commit new file.

Link to another website

It is totally ok if you just want to link to another website. You only need to put the metadata (at least the title, and the external URL) of the post between triple slashes as shown below.

--- title: "The Title of the post" urlex: "http://link/to/page.html" author:   name: John Doe   url: "https://johndoe.html" categories: [catA, catB] tags: [tagA, tagB] excerpt: A short description ---

Write a full post

If you plan to publish a full post directly on WASM.NEWS, you would just leave out the external URL. Instead you add the content of your post below the metadata section.

--- title: "The Title of the post" author:   name: John Doe   url: "http://johndoe.html" categories: [catA, catB] tags: [tagA, tagB] excerpt: A short description --- The body of a *full* blog post formated in [markdown](http://whatismarkdown.com/)

Thats all! I will try to merge every pull-request as soon as possible...
